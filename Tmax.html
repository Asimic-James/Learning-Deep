Front End Development

  • HTML
    • It's the foundation necessary in building a website, it lays a structure for a site.
  • Tags e.g <"....">
  • Elements e.g head, li, h2, body etc
  • CSS
    • Cascading Style Sheet Adds designs and colors to the site.
  • JavaScript
    • Is in charge of the functions and behaviors of the site. It also makes a site very dynamic and flexible in nature.

HTML Element is a parent element that contains other elements called Children.
Children of HTML- Header and Body

Header - Is initialized with the following element.."head" placed inside the "title" element..and it's not visible to the user but can only be seen at the task/title bar of any browser.

Body - The body displays the contents of the site to the user.
